# Bomber Prototype

Minimal Unity starter for a Bomberman-style game with:

- angled top-view camera inspired by MOBA framing
- smooth player movement on the XZ plane
- bomb placement, chained explosions, breakable crates
- simple roaming enemies
- power-up drops from destroyed crates
- lives, win/lose flow, restart prompt, and HUD
- runtime scene bootstrap so Play works without manual scene setup

## How to use

1. Open this folder as a Unity project.
2. Use a 3D template/project.
3. Press Play.

## Controls

- `WASD` or arrow keys: move
- `Space`: place standard bomb
- `Q`: place wall-breaker bomb
- `R`: restart after victory/defeat
